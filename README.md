# 2026년 전공분야 보수교육(융합) [IT직종]  
## **LangChain으로 만드는 웹 기반 챗봇 실습**

1. 프로젝트 주제 
	- 여행지 연관 추천 챗봇, 사용자의 기호와 이전 여행지의 정보를 기억하여 다음 여행지를 추천해주는 자동응답 시스템

2. 구현 결과물
	① GitHub 저장소 주소
		https://github.com/captainsix/langchain-260322

	② 또는 실행 파일(.zip) 제출 링크
	
4. 주요 기능 설명
	- 사용자가 여행지를 입력했을 때, 이전의 대화는 LCEL을 통해 메모리에 저장하고, 다음 질문에서는 이전의 내용을 요약하여 보여주는 챗봇 시스템

5. 실행 예시 및 결과 캡처	웹 실행화면 또는 동작 시나리오 이미지 첨부
	- image 폴더 안에 있음
	
5. 학습/구현 소감	프로젝트를 통해 배운 점, 개선 아이디어 등
	- LangChain을 사용하여 챗봇을 생성해 볼 수 있어 좋았습니다.
